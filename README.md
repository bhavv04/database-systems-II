# CPS610 - Database Systems II

Assignments covering advanced database concepts, from PL/SQL and object 
types to concurrency control, NoSQL, and distributed data processing.

## Assignments

### A2 - plsql-objects
Writing PL/SQL procedures and loops on Oracle DBMS. Covered user-defined 
object types, table creation from types, and procedural extensions to SQL 
including loops, procedures, and tax calculations.

### A3 - transaction-serializability
Simulating serializable schedules for concurrent transactions. Covered 
conflict-serializability testing, precedence graph construction, and 
detecting non-serializable schedules.

### A4 - concurrency-2pl
Implementing a Two-Phase Locking (2PL) algorithm for concurrent transaction 
management. Covered shared and exclusive locks, round-robin scheduling, 
system log tables for rollback, and deadlock detection via wait graphs.

### A5 - nosql-mapreduce
Installing and working with MongoDB for NoSQL CRUD operations. Covered 
document-based database architecture and implementing a Map-Reduce process 
on distributed text data using hash tables and linked lists to count 
movie occurrences across multiple data sources.
